# 字幕翻訳ワークフロー

SRT字幕ファイルを効率的に翻訳するための環境です。

## 📁 ディレクトリ構成
作業しやすいようにフォルダ分けされています。

- **`01_scripts/`**: ツール類（分割・結合などのPythonスクリプト）
- **`02_input/`**: **ここに翻訳したい元のSRTファイルを入れてください**
- **`03_output/`**: 翻訳完了後のファイルがここに保存されます
- **`99_temp/`**: 作業用の一時フォルダ
- **`00_docs/`**: ドキュメント類

---

## 🤖 Antigravityへの頼み方（推奨）

**一番簡単な方法**です。
SRTファイルを `02_input` フォルダに入れた状態で、チャットで以下のように伝えてください。

> **「`02_input` にある [元の言語] のファイルを、[翻訳したい言語] に翻訳して」**

Antigravityが内部で以下の処理を自動で行います。
1. ファイルの分割（必要な場合）
2. AIによる翻訳
3. 翻訳結果の結合（`03_output` フォルダに保存）

---

## 🛠 手動で行う場合（上級者向け）
自分でスクリプトを実行して翻訳する場合の手順です。

### 1. 準備 & 分割
長いSRTファイルはAIの入力制限を避けるため分割します。

```bash
# 02_inputフォルダ内のファイルを対象に実行
python3 01_scripts/split_srt.py
```

### 2. 翻訳
以下の「万能翻訳プロンプト」をコピーし、AIチャット（ChatGPT, Claude等）に貼り付けます。
続けて `99_temp/chunks/` (または `chunks/`) フォルダ内のファイルの中身を貼り付けて送信します。

#### 万能翻訳プロンプト
`[元の言語]` と `[ターゲット言語]` の部分を書き換えて使用してください。

```text
あなたはプロフェッショナルな映像翻訳家です。
以下の**[元の言語]**の字幕ファイル（.srt）の一部を、文脈を考慮して自然な**[ターゲット言語]**に翻訳してください。

## 制約事項
1. **フォーマット厳守**: 字幕番号（1, 2, 3...）とタイムスタンプ（00:00:00,000 --> ...）は**絶対に**変更・削除せず、原文のまま出力してください。
2. **翻訳の質**: 直訳ではなく、映像の字幕として読みやすく、会話の流れが自然な表現に意訳してください。
3. **文脈維持**: 前後の行のつながりを意識し、話者が話しているトーン（敬語、タメ口など）を原文のニュアンスに合わせて一貫させてください。
4. **分割処理**: 入力データが長い場合は、字幕番号30個程度ずつに分割して処理することを推奨します。

## 入力データ（SRT）
(ここにSRTデータを貼り付け)

## 出力形式
入力と同じSRTフォーマットで、テキスト部分のみを**[ターゲット言語]**に翻訳して出力してください。解説や前置きは不要です。
```

### 3. 保存
AIの出力結果を `99_temp/translated_chunks/` などに保存します。

### 4. 結合
翻訳済みファイルを一つにまとめます。

```bash
python3 01_scripts/combine_srt.py
```
`03_output` フォルダなどに完成ファイルが保存されます。
