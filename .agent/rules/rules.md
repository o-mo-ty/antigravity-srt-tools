# 字幕翻訳プロジェクト・ルール (rule.md)

このプロジェクトにおける字幕翻訳作業の共通ルールとガイドラインです。
AIエージェントおよび作業者は、以下のルールを厳守してください。

## 1. 翻訳の基本原則

- **フォーマット厳守**
  - SRT形式の厳密な維持が最優先です。
  - 字幕番号（Indices）とタイムスタンプ（例: `00:00:01,000 --> 00:00:04,000`）は、**一文字たりとも変更・削除してはいけません**。
  - 翻訳対象は「字幕テキスト」のみです。

- **翻訳品質**
  - **自然な会話文**: 直訳調を避け、映像の字幕として違和感のない、自然な日本語（またはターゲット言語）に意訳してください。
  - **文脈の維持**: 前後の字幕のつながりを常に意識してください。
  - **トーンの一貫性**: 話者のキャラクターや関係性（敬語、タメ口、呼び方など）を最初から最後まで統一してください。

## 2. ディレクトリ運用ルール

- **入力ファイル (`02_input/`)**
  - 翻訳元のSRTファイルは必ず `02_input` ディレクトリに配置してください。
  - ファイル名は変更せず、元のファイル名を維持することを推奨します。

- **出力ファイル (`03_output/`)**
  - 成果物は必ず `03_output` ディレクトリに保存します。
  - 中間ファイル（分割ファイルなど）は `99_temp` ディレクトリを使用し、ルートディレクトリを汚さないようにしてください。

## 3. ワークフロー

1. **ファイル配置**: ユーザーが翻訳元ファイルを `02_input` に置く。
2. **指示**: ユーザーが「[言語A]から[言語B]へ翻訳」を依頼。
3. **処理**:
   - 長いファイルは `99_temp` 内で分割。
   - 分割ファイルを翻訳。
   - 翻訳済みファイルを結合。
4. **完了**: 結合されたファイルを `03_output` に保存し、ユーザーに報告。
